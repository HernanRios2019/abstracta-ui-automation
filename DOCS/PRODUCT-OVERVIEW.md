# Product overview

## Software solution:
The solution is a Software as a Service (SaaS) platform that features a tenancy website. Users can access either the Admin view or the Platform view, depending on the access permissions defined by their assigned role.

Users can log in using their email and password or via Single Sign-On (SSO), based on the client’s configuration. Once logged in, they are redirected to the Home page, where they can see their assigned workspaces and have buttons to access either the Admin or Platform view.

In the Admin view, administrators can create and manage documents and processes. In the Platform view, both end users and admins can view and interact with documents and processes, depending on the permissions associated with their assigned role.
